# 自定义 Electron 应用图标

生成图标。建议先从 `1024 x 1024px` 的图像开始，然后再将其转换为各种尺寸。

## 支持的格式

各平台推荐的文件格式及图标大小如下：

|操作系统 | 格式 | 尺寸|
---- | ---- | ----
Mac OS | `.icns` | 512x512像素（Retina 显示屏为 1024x1024）
Windows | `.ico` | 256x256像素
Linux | `.png` | 512x512像素
