## git stash summary

1. `git stash save "save message"`: 执行存储时，添加备注，方便查找，只有 `git stash` 也可以，但是查找时不方便。
2. `git stash list`: 查看所有存储的快照。
3. `git stash show stash@{0}`: 查看指定存储快照的内容。
4. `git stash show -p stash@{0}`: 查看指定存储快照的内容。
5. `git stash apply stash@{0}`: 应用指定存储快照。
6. `git stash drop stash@{0}`: 删除指定存储快照。
7. `git stash pop stash@{0}`: 应用存储快照，并删除。
8. `git stash clear`: 删除所有存储快照。
9. `git stash create`: 创建一个存储快照。
10. `git stash branch <branchname> stash@{0}`: 从指定存储快照创建一个分支。
